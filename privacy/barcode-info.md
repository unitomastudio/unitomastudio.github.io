# Barcode Info プライバシーポリシー

unitomastudio（以下「当方」）は、Androidアプリケーション「Barcode Info」（以下「本アプリ」）におけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

## 1. 収集する情報

### 1.1 端末内に保存される情報（外部送信なし）

本アプリは、以下の情報を端末内のみに保存します。

- スキャンしたバーコードの値
- バーコードのフォーマット、桁数、チェックデジット検証結果等の解析結果
- 印刷品質の評価結果

これらの情報は、お使いの端末内（ブラウザの localStorage）に保存され、当方を含む第三者に送信されることはありません。本アプリをアンインストールすると、これらの情報は消去されます。

### 1.2 カメラ

本アプリは、バーコードのスキャンを目的としてカメラを使用します。撮影された画像は端末外に送信・保存されることはなく、バーコード読み取りおよび印刷品質評価（コントラスト・サイズ等）のために端末内でのみ使用されます。

### 1.3 外部サービスへの送信

**本アプリは完全にオフラインで動作し、いかなる外部サービスにもデータを送信しません。**

- バーコード解析は端末内の Web BarcodeDetector API で実行
- 広告配信なし
- アナリティクスなし

## 2. 第三者への提供

当方は、ユーザーの個人情報を販売または第三者に提供することはありません。

## 3. 子ども（13歳未満）について

本アプリは開発者向けのツールで、13歳未満を主たる対象としていません。
